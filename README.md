# 3D Portfolio Website

## Introduction
This website gives a prospective client or employer a quick look at me and my work as a developer, showing them what I can do with not just the projects listed, but the website itself. And yes, I know the parallax gets tricky when shrinking the window, but I'll fix that... one day... maybe.

## Technologies
- Vite - Build tool for the boilerplate and structure
- JavaScript - Programming language
- React - JavaScript library for building user interfaces
- Tailwind - CSS framework
- Three.js - Animated 3D graphics
- Framer Motion - Interactive animations
- Vercel - Hosting the website, you can view it live at https://navalshah.vercel.app/
